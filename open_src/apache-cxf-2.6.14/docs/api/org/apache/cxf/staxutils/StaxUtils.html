isWoodstox

public static boolean isWoodstox()
CXF works with multiple STaX parsers. When we can't find any other way to work against the different parsers, this can be used to condition code. Note: if you've got Woodstox in the class path without being the default provider, this will return the wrong answer.

Returns:
true if Woodstox is in the classpath.
